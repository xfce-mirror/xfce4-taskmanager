### Installation

From source code repository: 

    % cd xfce4-taskmanager
    % meson setup build
    % meson compile -C build
    % meson install -C build

From release tarball:

    % tar xf xfce4-taskmanager-<version>.tar.xz
    % cd xfce4-taskmanager-<version>
    % meson setup build
    % meson compile -C build
    % meson install -C build

### Uninstallation

    % ninja uninstall -C build
